# 2048

## About the game

Classic 2048 game 

<br>

## Try it

- Download the repo
- Open terminal and navigate to where the folder is (ie. "cd ~/downloads/2048")
- Type "npm install" or yarn install
- Type "npm start"
- Open http://localhost:4200/
